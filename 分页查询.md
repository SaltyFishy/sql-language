# sql-learning in week10(using MySQL)
<h2>分页查询</h2>
使用情况：数据量过于庞大的时候（一个页面无法全部显示），对数据进行分页，用户需要的时候打开该页面<br>
语法：

```mysql
select 查询列表 
from 表1 
[[join type] join 表2]
[on连接条件]
[where 筛选条件]
[group by 分组条件]
[having 分组后筛选]
[order by 排序字段 排序]
limit offset,size; 
#offset为起始索引(起始索引为0时可不写)，size为要显示的条目数
#起始索引从0开始
```
<br>
例1：查询前五条员工信息

```mysql
SELECT * 
FROM employees 
LIMIT 0,5;
```
<br>

例2：查询11-25条员工信息

```mysql
SELECT *
FROM employees
LIMIT 10,15;
```
<br>

例3：查询有奖金的员工信息并且显示前十名工资

```mysql
SELECT *
FROM employees
WHERE commission_pct IS NOT NULL
ORDER BY salary DESC
LIMIT 10;
```
