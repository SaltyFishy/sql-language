# sql-learning in week10(using MySQL)
<h2>联合查询</h2>
涉及关键字：union<br>
作用：将多条查询语句的结果合并为一个结果<br>
适用范围：当查询多个表并且多个表之间没有联系时<br>
例如：在网页进行查询信息时输入多个关键字，网站会查询每个关键字的信息，最后组合<br>
<br>
<strong>要求多条查询语句的查询列数一致</strong><br>
<strong>多条查询语句的多条查询字段的实际意义与顺序一致</strong><br>
<strong>union关键字会自动去重，如果不想去重可以使用union all</strong><br>
引入：查询部门编号大于90或邮箱中包含a的员工信息
以前的方法：

```mysql
SELECT * 
FROM employees
WHERE email LIKE '%a%' AND department_id > 90;
```
<br>

联合查询：

```mysql
SELECT * FROM employees WHERE email LIKE '%a%' 
UNION 
SELECT * FROM employees WHERE department_id > 90;
```
<br>
语法：

```mysql
查询语句1
UNION
查询语句2
...
```
<br>
例1：查询中国用户中性别为男的用户信息与外国用户中男性用户信息

```mysql
SELECT * FROM t_ca WHERE csex='男'
UNION
SELECT * FROM t_ua WHERE tGender='男'
```
